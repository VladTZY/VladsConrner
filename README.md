# blog coneschi - client

frontend for vlad coneschi's review blog.

## stack

-   next.js 14
-   tailwind css
-   typescript

## design

-   **minimalist**: clean lines, whitespace.
-   **typography**: `outfit` (sans) and `space mono` (mono).
-   **vibe**: coffee shop / gen z. casual, lowercase.

## getting started

1.  `cd client`
2.  `npm install`
3.  `npm run dev`
